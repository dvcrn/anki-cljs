# anki-cljs

ClojureScript interface for interacting with Anki

Pretty much in progress

## Hack

Start the nREPL
```
lein repl

(use 'figwheel-sidecar.repl-api)
(start-figwheel!)
```

Connect to it and run
```
(use 'figwheel-sidecar.repl-api)
(cljs-repl)
```
